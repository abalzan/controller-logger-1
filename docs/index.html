A Java library to ease the life of web developers by providing automatic logging of input-output for controllers.
Logged items include all parameters, returned value and some context data such as web request URL and user's username.

Automatically logs all APIs including input and output Automatically logs errors occurring in API No side-effect in actual API implementation due to AOP logic Automatically binds to new APIs thanks to AOP weaving Scrubs sensitive information in logs to maintain security and privacy Displays file size if one of the API input or output is any file object Works with integration testing Detects mocked objects in input and output and displays them accordingly, as can happen during integration testing Logging behaviour is easily customizable
